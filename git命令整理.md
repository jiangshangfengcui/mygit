1、创建新文件 touch + 新建文件名
	
2、创建并编辑新文件 vi + 新家文件名

3、创建目录   mkdir + 新建目录名

4、切换目录   cd (change directory)

5、删除文件   git rm ( 删除的是已跟踪的文件 )

6、移动文件  git mv source destination 移动的文件必须是已跟踪的文件
   
7、清屏       reset 或者 clear 

8、退出编辑   : + q + Enter

9、强制退出编辑 : + q! + Enter

10、退出编辑并保存 : + x + Enter

11、删除当前行     dd

12、切换到编辑模式    i

13、退出编辑模式    Esc + : + x + Enter
